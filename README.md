A fun pygame game where animals bungee jump! Built with Python and pygame, running in the browser via pygbag.

## Play Now

Visit [bungeezoo.com](https://bungeezoo.com) to play!

## Controls

- **Click/Tap** - Reset all animals
- **R key** - Reset all animals
- **ESC** - Exit (desktop only)

## Built With

- [pygame-ce](https://pyga.me/) - Python game library
- [pygbag](https://pygame-web.github.io/) - Run pygame in the browser
